# Система мониторинга датчиков для объектов атомной энергетики

Дипломный проект (специальность 09.02.07 «Информационные системы и программирование»). Решение для удалённого мониторинга показаний датчиков на промышленных объектах атомной энергетики, разработанное в контексте задач Росатома.

## Описание

Проект состоит из двух частей:

- **Backend** — серверная часть на FastAPI, отвечающая за приём, хранение и обработку данных с датчиков через REST API.
- **Mobile-приложение** — клиент на Kotlin/Android для просмотра показаний датчиков в реальном времени и контроля состояния оборудования.

Архитектура построена так, чтобы данные с физических датчиков поступали на сервер, сохранялись в базе данных и были доступны для просмотра через мобильное приложение в удобном виде, с возможностью оперативно отслеживать отклонения от нормы.

## Технологический стек

**Backend:**
- Python 3 / FastAPI
- MySQL
- REST API

**Mobile:**
- Kotlin
- Android SDK

## Установка и запуск

### Backend

```bash
cd backend
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt

# настроить переменные окружения для подключения к MySQL (.env)

uvicorn main:app --reload
```

### Mobile-приложение

1. Открыть папку `mobile` в Android Studio.
2. Указать адрес backend-сервера в конфигурации приложения.
3. Собрать и запустить на эмуляторе или физическом устройстве.
